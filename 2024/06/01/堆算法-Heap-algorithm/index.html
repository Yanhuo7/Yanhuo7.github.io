1.Introdution

Heap is a kind of nolinear data structure(complete binary tree) that can be divied into two types，max heap and mini heap.

Max heap : any node value >= it’s child node value , root node has the largest value ;

Min heap: any node value <= it’s child node value ,root node has the smallest value;
